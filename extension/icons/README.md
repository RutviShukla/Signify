# Extension Icons

This directory should contain the following icon files:

- `icon16.png` - 16x16 pixels (toolbar icon)
- `icon48.png` - 48x48 pixels (extension management page)
- `icon128.png` - 128x128 pixels (Chrome Web Store)

## Quick Icon Creation

You can create simple placeholder icons using any image editor:

1. Create square images with the specified dimensions
2. Use a gradient background (purple/blue theme: #667eea to #764ba2)
3. Add "DF" or "Deaflix" text in white
4. Save as PNG files

## Online Tools

- Canva (free icon templates)
- Figma (design tool)
- GIMP (free image editor)
- Photopea (online Photoshop alternative)

## For Demo

For the hackathon demo, you can use simple colored squares or download free icons from:
- Flaticon
- Icons8
- Font Awesome

Make sure icons are recognizable and match the Deaflix brand colors.
