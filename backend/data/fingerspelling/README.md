# Fingerspelling Images

Place letter images (A-Z) and number images (0-9) in this directory.

## File Naming

- Letters: `a.png`, `b.png`, `c.png`, ..., `z.png`
- Numbers: `0.png`, `1.png`, `2.png`, ..., `9.png`

## Format

- PNG format recommended
- Square aspect ratio (e.g., 200x200px)
- Clear hand signs for each letter

## Alternative

If you don't have fingerspelling images, the system will fall back to:
- Letter images from `data/asl_dataset/{letter}/` folders
- Using existing ASL dataset letter images

## Quick Setup

You can create placeholder images or use existing letter images from your ASL dataset.
