# WLASL Setup Scripts

## Scripts Overview

### 1. `generate-wlasl-mapping.js`

Generates a clean `mapping.json` file from WLASL dataset metadata.

**Usage:**
```bash
cd backend
node scripts/generate-wlasl-mapping.js
```

**Requirements:**
- `data/WLASL_v0.3.json` - WLASL metadata file
- `data/missing.txt` (optional) - List of missing video IDs
- `data/WLASL_videos/` - Directory with video files
- `data/mvp_vocab.json` - Top 200 vocabulary list

**Output:**
- `data/mapping.json` - Clean mapping of gloss → video files

### 2. `copy-wlasl-videos.js`

Copies WLASL videos to organized folder structure.

**Usage:**
```bash
cd backend
node scripts/copy-wlasl-videos.js
```

**Requirements:**
- `data/mapping.json` - Generated by `generate-wlasl-mapping.js`
- `data/WLASL_videos/` - Source video directory

**Output:**
- `data/asl_dataset/{gloss}/{video_id}.mp4` - Organized video files

## Setup Workflow

1. **Download WLASL Dataset**
   - Get `WLASL_v0.3.json` from WLASL repository
   - Download video files to `data/WLASL_videos/`

2. **Generate Mapping**
   ```bash
   node scripts/generate-wlasl-mapping.js
   ```

3. **Copy Videos**
   ```bash
   node scripts/copy-wlasl-videos.js
   ```

4. **Restart Backend**
   ```bash
   npm start
   ```
