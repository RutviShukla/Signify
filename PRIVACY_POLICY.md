# Signify Privacy Policy

**Last Updated:** November 2024

## Introduction

Signify is a Chrome extension that provides real-time American Sign Language (ASL) interpretation for YouTube videos, making video content accessible to deaf and hard-of-hearing users.

## Data Collection

### What We Collect

**Website Content:**
- The extension reads YouTube video captions from pages you visit
- Caption text is sent to our backend API to retrieve corresponding ASL video files
- Caption text is processed in real-time and is not stored permanently

**Local Storage:**
- User preferences (ASL widget visibility, caption display settings, widget positions)
- Backend URL configuration
- All preferences are stored locally on your device using Chrome's storage API

### What We Do NOT Collect

- Personal information (name, email, address)
- Health information
- Financial information
- Authentication credentials
- Location data
- Web browsing history
- User activity tracking
- Any data that identifies you personally

## How We Use Your Data

**Caption Text:**
- Sent to our backend API to match words with ASL videos
- Processed in real-time and not stored
- Used solely to provide ASL interpretation functionality

**Local Preferences:**
- Stored on your device only
- Used to remember your settings between sessions
- Never transmitted to external servers

## Data Sharing

- We do NOT sell your data
- We do NOT share your data with third parties
- Caption text is only sent to our own backend API for ASL video matching
- No data is transferred to any other services or companies

## Data Storage

- All user preferences are stored locally on your device
- Caption text is processed in real-time and not stored
- Our backend API does not log or store caption text permanently

## Security

- All API communications use HTTPS encryption
- No sensitive personal information is collected or transmitted
- Local storage is managed by Chrome's secure storage API

## Third-Party Services

- **Railway (Backend Hosting):** Our backend API is hosted on Railway. Railway's privacy policy applies to infrastructure-level data only. We do not share user data with Railway beyond what is necessary for API hosting.

## Your Rights

- You can disable the extension at any time
- You can clear stored preferences through Chrome's extension settings
- No account is required, so there is no data tied to your identity

## Changes to This Policy

We may update this privacy policy from time to time. The "Last Updated" date at the top indicates when changes were made.

## Contact

For questions about this privacy policy, please contact us through the Chrome Web Store listing.

## Compliance

This extension complies with:
- Chrome Web Store Developer Program Policies
- General Data Protection Regulation (GDPR) principles
- California Consumer Privacy Act (CCPA) principles
